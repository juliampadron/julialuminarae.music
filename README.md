# Julia Lumina Rae Music

A Brutalist VHS Neo-Noir artist landing page for Julia Lumina Rae, built with React, Vite, Tailwind CSS, and custom audio preview interactions.

## What is included

This site presents Julia Lumina Rae as an independent artist across music, writing, psychic mediumship, and Hudson Valley community projects. It includes a glitch-forward hero section, a featured singles grid with custom audio previews, album artwork, donation/support calls to action, YouTube linking, social links, and an honor-system download unlock flow.

## Notes

Large media files are referenced through hosted storage paths rather than committed directly to the repository. The site uses an honor-system support flow for downloads and links to Cash App at `$feedthegemini`.
